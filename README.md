# 내 주식 포트폴리오 (2개 증권사 통합 관리)

매입가는 직접 입력하고, 현재가는 **한국투자증권 KIS Open API**로 자동 조회해
A·B 두 증권사의 평가손익을 각각, 그리고 합산해서 보여주는 웹사이트입니다.

---

## 왜 "서버"가 필요한가요? (비개발자용 설명)

증권사 시세 API는 **웹 브라우저에서 직접 부를 수 없습니다.** 두 가지 이유 때문입니다.

| 문제 | 쉬운 설명 |
|---|---|
| CORS 차단 | 은행 창구가 "우리 직원(서버)을 통해서만 업무 가능"이라고 막아둔 것과 같습니다. 브라우저가 KIS에 직접 요청하면 거절당합니다. |
| 시크릿 노출 | API 시크릿(비밀번호)을 웹페이지에 넣으면 누구나 볼 수 있어 도용될 수 있습니다. |

그래서 아주 작은 **중계 서버(프록시)** 를 하나 두고, 브라우저 → 내 서버 → KIS 순서로 호출합니다.
`api/price.js` 파일이 그 중계 서버 역할을 하며, 시크릿은 서버에만 안전하게 보관됩니다.
이 구조를 무료로 배포하는 가장 쉬운 방법이 **Vercel** 입니다.

## 전체 순서 한눈에 보기

| 단계 | 할 일 | 예상 시간 |
|---|---|---|
| 1 | 한국투자증권 KIS 앱키·앱시크릿 발급 | 10분 |
| 2 | Vercel 가입 후 이 폴더 업로드(배포) | 5분 |
| 3 | Vercel에 API 키 2개를 환경변수로 등록 | 3분 |
| 4 | 사이트 주소 접속 → "현재가 새로고침" 확인 | 1분 |

---

## 1단계 · KIS 앱키 발급

| 순서 | 내용 |
|---|---|
| 1-1 | 한국투자증권 **계좌**가 필요합니다 (모바일 앱 등으로 개설 가능). |
| 1-2 | KIS Developers 접속 → https://apiportal.koreainvestment.com |
| 1-3 | 로그인 후 **"KIS Developers 신청"** → 실전투자(또는 모의투자) 선택 후 신청 |
| 1-4 | 발급된 **App Key** 와 **App Secret** 을 복사해 둡니다. (2단계에서 사용) |

> 참고: API 이용료는 무료이며, HTS 이용료에 포함됩니다.
> 모의투자 계정으로 발급했다면 `api/price.js` 파일의 `BASE` 주소를 모의투자 도메인으로 바꿔야 합니다 (파일 상단 주석 참고).

---

## 2단계 · Vercel로 배포

| 순서 | 내용 |
|---|---|
| 2-1 | https://vercel.com 에 가입 (GitHub/구글 계정으로 간편 가입) |
| 2-2 | 가장 쉬운 방법: Vercel CLI 없이 **드래그 앤 드롭** 또는 GitHub 연동 |
| 2-3 | **방법 A (GitHub):** 이 `stock-portfolio` 폴더를 GitHub 저장소에 올리고, Vercel에서 "New Project → Import" |
| 2-4 | **방법 B (CLI):** 터미널에서 이 폴더로 이동 후 `npx vercel` 실행하고 안내를 따라가면 됩니다 |
| 2-5 | 배포가 끝나면 `https://내프로젝트이름.vercel.app` 주소가 생깁니다 |

> `index.html`(화면)과 `api/price.js`(중계 서버)가 한 폴더에 있어, 별도 설정 없이 자동으로 함께 배포됩니다.

---

## 3단계 · API 키를 환경변수로 등록 (중요)

시크릿을 코드에 넣지 않고 Vercel에 안전하게 보관하는 단계입니다.

| 순서 | 내용 |
|---|---|
| 3-1 | Vercel 프로젝트 → **Settings → Environment Variables** |
| 3-2 | 이름 `KIS_APP_KEY`, 값 = 1단계의 App Key 입력 후 저장 |
| 3-3 | 이름 `KIS_APP_SECRET`, 값 = 1단계의 App Secret 입력 후 저장 |
| 3-4 | **Deployments → 최신 배포 → Redeploy** (환경변수를 반영하려면 재배포 필요) |

---

## 4단계 · 사용하기

| 항목 | 설명 |
|---|---|
| 종목 추가 | 각 증권사 표 아래 **"+ 종목 추가"** 클릭 |
| 입력 항목 | 종목명 / 종목코드(6자리) / 보유수량 / 매입평균가 |
| 현재가 | 상단 **"🔄 현재가 새로고침"** 클릭 시 KIS API로 자동 조회 |
| 자동 계산 | 매입금액·평가금액·평가손익·수익률, 증권사별 소계, A+B 합산 모두 자동 |
| 저장 | 입력값은 접속한 브라우저에 자동 저장됩니다 |
| 증권사 이름 | 상단의 "A증권" 글자를 클릭해 실제 증권사명으로 변경 가능 |

> **시세 서버 연결 전에도 사용 가능:** 배포·키 설정 전이라면 현재가 칸을 클릭해 직접 입력하면 손익은 그대로 자동 계산됩니다.
